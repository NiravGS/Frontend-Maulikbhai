# Project Setup Guidelines

## Prerequisites

- [Node v10.0+](https://nodejs.org/en/download/current/) or [Docker](https://www.docker.com/)

- [npm v6.0+](https://www.npmjs.com)

## Getting Started

1. Clone the repo :

```bash
git clone https://positive-proton@bitbucket.org/positive-proton/expeditions-frontend-redesign.git expeditions-frontend
cd expeditions-frontend
rm -rf .git
```

2. Install dependencies:

```bash
npm install
```

## Running Locally

```bash
npm run start
```

## Making Production Build

```bash
npm run build
```
